📊 Análisis de desempeño de proveedores – Estampería y Bordado

DESCRIPCION DEL PROYECTO

Este proyecto analiza el desempeño de proveedores dentro del proceso de desarrollo de muestrario para colecciones de indumentaria.
El análisis compara dos temporadas: Verano 26 (V26) e Invierno 26 (I26), con el objetivo de identificar patrones en volumen de artículos, tiempos de desarrollo y tasa de aprobación de muestras.
A partir de estos datos se busca detectar atrasos en el proceso, participación de proveedores y posibles cuellos de botella en el desarrollo de producto.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

PREGUNTAS DE ANALISIS

El dashboard fue diseñado para responder las siguientes preguntas:
1) ¿Qué volumen de artículos se trabajó en cada temporada?
2) ¿Qué proveedores presentan mayores tiempos de desarrollo?
3) ¿Cómo cambia la participación de los proveedores entre temporadas?
4) ¿Qué artículos registraron los mayores retrasos en el proceso?

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

METRICAS ANALIZADAS

El análisis se basa en tres indicadores principales:

1) Cantidad de artículos desarrollados por temporada
2) Promedio de días de desarrollo de muestrario
3) Tasa de aprobación de muestras

Estos indicadores permiten analizar el desempeño de los proveedores y detectar posibles oportunidades de mejora en el proceso.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

DASHBOARD

El tablero permite visualizar de forma interactiva:

1) Comparación de KPIs entre temporadas
2) Ranking de proveedores según tiempos de desarrollo
3) Participación de proveedores por temporada
4) Artículos con mayores retrasos en el proceso
